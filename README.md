# Visualizador de Perfil do GitHub

Este é um projeto de um visualizador de perfil do GitHub, que permite aos usuários pesquisar por um nome de usuário e ver informações do perfil, como nome, bio, número de seguidores e repositórios.

## Como Usar

1.  Abra o arquivo `index.html` em seu navegador.
2.  Digite um nome de usuário do GitHub na caixa de pesquisa.
3.  Clique no botão "Buscar" ou pressione Enter.
4.  As informações do perfil do usuário e seus repositórios serão exibidas.

## Tecnologias Utilizadas

*   HTML
*   CSS
*   JavaScript
*   [GitHub API](https://docs.github.com/en/rest)
